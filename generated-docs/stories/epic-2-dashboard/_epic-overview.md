# Epic 2: Dashboard (Screen 1)

## Description
Build the Dashboard screen — the application's home page — with six visual components showing commission payment metrics, an Agency Summary grid for navigation, agency-level filtering, and role-based pre-filtering. This epic delivers Screen 1 of the BetterBond Commission Payments POC.
